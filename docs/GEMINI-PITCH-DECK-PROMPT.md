## Usage tips

1. **One slide at a time** — If quality drops, ask Gemini: *"Generate slide 1 only, then we'll continue."*
2. **Google Slides export** — *"Export as editable Google Slides, then I'll download PDF."*
3. **Replace site deck** — Save output as `public/trvrse-pitch-deck.pdf` for the landing page download button.
4. **Local fallback** — Regenerate the basic text-only PDF anytime:

```bash
npm run generate:deck
```
