# Trvrse Official Brand Assets

Production logos from the Nexxogen brand identity board (Gemini vector recreation).

## Primary files (use these)

| File | Source | Use |
|------|--------|-----|
| `trvrse-logomark.png` | Isolated T-arrow mark (transparent) | Navbar, footer, UI |
| `trvrse-app-icon.png` | Squircle app icon (transparent) | App Store, PWA, `app/icon.png` |
| `trvrse-lockups-navy.png` | Horizontal + stacked lockups on navy | Decks, marketing |
| `trvrse-brand-board.png` | Full identity delivery sheet | Investors, brand guidelines |

## In code

```ts
import { BRAND } from "@/lib/branding";
// BRAND.logomark → /branding/trvrse-logomark.png
```

## Legacy / generated

Older SVG exports and `Gemini_Generated_Image_*.png` originals are kept for reference. Prefer the canonical `trvrse-*.png` files above.
